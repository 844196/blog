## Requirements

- Ruby (>=2.1.0)
- bundler
- lessc (for less compile)


## Local Setup

```shellsession
$ bundle install --path vendor/bundle
```


## Live Preview

```shellsession
$ bundle exec guard start
```

**note:** Require [chrome plugin](https://chrome.google.com/webstore/detail/livereload/jnihajbhpnppcggbcgedagnkighmdlei).
